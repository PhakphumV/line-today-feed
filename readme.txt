Send Feed to Line Today Application

== Description ==

Just upload this plugin to WordPress and activate it. The plugin will create rss feed for line today application.

Official development of Line Today feed is on GitHub, with official stable releases published on WordPress.org. The GitHub repo can be found at https://github.com/tannysoft/line-today-feed. Please use the Support tab for potential bugs, issues, or enhancement ideas.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/line-today-feed` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
